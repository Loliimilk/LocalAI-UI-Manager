# LocalAI-UI-Manager
**LocalAI-UI-Manager** — это десктоп-приложение на Rust, которое анализирует характеристики вашего компьютера, автоматически подбирает, скачивает и настраивает подходящую большую языковую модель (LLM). Всё это — с удобным графическим интерфейсом для управления моделями и общения с ними. 
> ⚠️ **Статус проекта:** Проект находится в активной разработке (альфа-версия). Реализованы еще не все функции, но базовый функционал уже доступен для тестирования.
> **Текущее состояние:**
> - Реализовано: сканер железа и система сбора данных в единую структуру
> - В процессе: рекомендация по улучшению производительности и установка полезных зависимостей
> - Остальные функции в разработке
## Возможности программы 
- Сканирует систему пользователя для определения характеристик железа.
- Скачивает зависимости, необходимые для оптимальной работы LLM.
- Подбирает подходящую модель LLM под мощности вашей системы.
- Производит автоматическую настройку LLM под конкретную систему.
- Предоставляет удобный интерфейс для настройки параметров модели.
- Включает интерфейс для общения с LLM.
- Предоставляет API для интеграции с другими сервисами.
