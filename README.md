# LocalAI-UI-Manager
LocalAI-UI-Manager - десктоп-приложение на Rust для автоматического подбора, установки и запуска больших языковых моделей (LLM) локально на вашем компьютере, на основе анализа характеристик компьютера пользователя. Программа предоставляет графический интерфейс для управления моделями и общения с ними.
